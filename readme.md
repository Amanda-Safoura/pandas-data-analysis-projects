# Pandas Data Analysis Projects

Collection of data analysis projects completed while learning Pandas, NumPy fundamentals, and data visualization.

## Objectives

This repository focuses on:

- Data loading
- Data cleaning
- Data filtering
- Exploratory Data Analysis (EDA)
- Data visualization
- Simulation

---

## Projects

### 1. Netflix Dataset Analysis

Analysis of a movie and TV show dataset using Pandas.

#### Features

- Load CSV datasets
- Explore dataset structure
- Filter movies and TV shows
- Analyze release years
- Analyze countries and genres
- Create visualizations

#### Concepts Practiced

- Pandas DataFrames
- read_csv()
- loc[]
- iloc[]
- Boolean filtering
- Aggregations
- Matplotlib

---

### 2. Random Walk Simulation

Simulation of a random walk process using NumPy and Matplotlib.

#### Features

- Generate random steps
- Simulate multiple walks
- Compute probabilities
- Analyze outcomes
- Visualize trajectories
- Plot distributions

#### Concepts Practiced

- Random number generation
- Loops
- Simulations
- Probability
- Histograms
- Line plots
